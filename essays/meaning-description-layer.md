# 意味のレントゲン ― Sens が LLM の「外側」に置く記述レイヤー

## 1. 見えないまま、うまくいってしまう

大規模言語モデルとの対話には、奇妙な成立条件がある。出力は読める。
やりとりは続く。多くの場合、ちゃんと「うまくいって」しまう。にも
かかわらず、そこで何が運ばれ、何が落ち、何が静かにすり替わったのか
は、ほとんど見えない。

見えないまま成立する、というのは厄介な状態だ。失敗していれば人は
立ち止まる。だが成功しているように見えるとき、人は問いを持たない。
意味の取り扱いについて、私たちは長らくこの「成功による盲目」の中に
いた。

Sens はこの盲目を主題にする。テキストをより上手に処理する道具では
ない。テキストの「下」を流れている意味そのものを、記述可能な対象に
する ―― LLM の外側に置かれた、意味の記述レイヤーである。

## 2. 土俵を疑う

意味の取り扱いをめぐる既存のアプローチ ―― 長い文脈窓、ベクトル
検索による参照、グラフ構造での連結、手順書的なスキル定義 ―― は、
互いに competitor のように見える。だが一段引いて見ると、これらは
同じ一つの土俵を共有している。**テキストを、より大きく・より速く・
より構造的に出し入れする**、という土俵だ。

土俵が同じなら、勝敗は程度問題になる。もっと長い窓、もっと良い
検索、もっと精緻なグラフ。改善は積み上がるが、地平は変わらない。
そして決定的なことに ―― そのすべてが、意味を「テキストとして」
しか扱っていない。意味は依然として、文字列の振る舞いから事後的に
推測されるものであり続ける。

Sens の最初の主張は、機能の優劣ではない。**この土俵に乗らない**、
である。テキスト操作の精度を競うのではなく、意味そのものを一次的な
記述対象として切り出す。問いを「どうテキストを動かすか」から
「意味をどう記述し、見て、運ぶか」へ移す。これは改善ではなく、
カテゴリの移動だ。

## 3. LLM を VM とみなす

カテゴリの移動を一つのアナロジーで言い直す。**LLM を仮想マシン
(VM) とみなす**、というものだ。

VM は強力な実行基盤だが、それ自体は「何を実行すべきか」を語らない。
意図は外から与えられ、コンパイルされ、ロードされる。VM の内部状態を
直接書くのではなく、より上の層で記述したものを、実行可能な形へ
落とす ―― これがソフトウェアの常識的な分業だ。

LLM に対して、私たちはこの分業をまだ持っていない。意図も、文脈も、
過去の合意も、すべてを生のテキストとして VM に流し込み、VM の振る
舞いを見て事後的に「通じたらしい」と判断している。アセンブリを
手書きし、実行結果から仕様を逆算しているようなものだ。

Sens が置こうとするのは、その欠けた層 ―― **意味のコンパイラ層**で
ある。人が扱うのは意味の記述であり、それを LLM という VM が解せる
形へ橋渡しする。LLM を賢くする話ではない。LLM の外側に、意味を
扱うための上位レイヤーを一段、足す話だ。

## 4. 四つの操作

レイヤーが実在すると言えるのは、その上で固有の操作が定義できるとき
だ。Sens が意味記述レイヤーの上で立てる操作は、さしあたり四つ。

**(1) 意味の Diff。** テキストの差分ではなく、意味の差分を取る。
語が変わっても意味が同じこと、語が同じでも意味がずれたこと ――
文字列比較では原理的に見えないこの二つを、記述レイヤーの上では
差分として扱える。

**(2) 意味の選別注入。** 文脈に「全部入れる」のでも「検索で似た
ものを足す」のでもなく、いま必要な意味だけを選んで注入する。
対象は文書ではなく、意味の単位である。

**(3) 意味のモデル間永続化。** あるモデルとの間で築いた意味を、
別のモデルへ持ち越す。テキストのコピーではない ―― モデルが替わ
っても保たれることを意図した、意味そのものの持続だ（この可能性の
理論的背景は §6 で述べる）。

**(4) 意味の可視化。** 推測するのでも要約するのでもなく、意味の
構造をそのまま見る。これが次節の中心的なメタファーに繋がる。

四つは別々の機能の寄せ集めではない。「意味を一次的対象として扱う」
という同じ前提から導かれる、同じレイヤー上の四つの動詞である。

## 5. 意味のレントゲン

可視化を、機能名としてではなく一つの像として言いたい。**意味の
レントゲン、あるいは CT スキャン**である。

レントゲンの意義は「写真がきれいになること」ではない。**通常は
原理的に見えないものを、見えるようにしたこと**にある。骨折は以前
から存在していた。患者の訴えから推測することもできた。だがレント
ゲンは推測を観察に変えた。それは既存技術の改良ではなく、診断と
いうカテゴリそのものの再定義だった。

意味も同じ位置にある。LLM との対話で運ばれる意味は、ずっとそこに
あった。私たちは振る舞いから推測してきた ―― 訴えから骨折を推し
量るように。Sens の可視化が主張するのは「推測の精度を上げる」こと
ではなく、**推測を観察に置き換える**ことだ。だから比較軸は機能
ではない。レントゲン以前か以後か、である。

## 6. 推測を観察に変えてよい理由

この主張には根拠が要る。意味が記述・観察・持ち越し可能な構造を
本当に持つ、と言えるのはなぜか。

一つの公刊された手がかりは、**Platonic Representation Hypothesis**
(MIT CSAIL, 2024) だ。異なるデータ・異なる目的で訓練された別々の
モデルが、内部表現において収束していく傾向が観察される ―― あたかも
モデル非依存の、共有された世界の構造へ近づくかのように、という
仮説である。これが含意するのは重い。意味が各モデルのローカルな癖に
すぎないなら、観察も持ち越しも幻想だ。だが意味にモデルを超えた構造が
あるなら、**それは記述しうる対象であり、モデル間で永続化しうる対象
でもある**（§4 の操作 (3) と (4) の前提が、ここに接続する）。

もう一つは、解釈可能性研究の系譜 ―― とりわけ **sparse autoencoder**
に代表される、内部表現を人が辿れる単位へ分解する公刊手法の蓄積で
ある。Sens はこれら個別手法のどれかではない。だがその系譜は、
「内部に流れているものは原理的に不可視ではない」という一点を、
研究コミュニティの公開された成果として支えている。レントゲンが
比喩で終わらないための、外部からの裏付けだ。

（注：本節は公刊文献への一般的言及にとどめる。内部の実装・実験・
数値はこのレイヤーの議論には不要であり、ここでは扱わない。）

## 7. 重さ ―― 儀式性としての

ここから哲学に入る。技術の話を一段降りて、なぜこの可視化が単なる
便利さ以上のものなのかを言う。

LLM との対話で、意味は**軽く**なった。即座に生成され、摩擦なく
流れ、惜しまれずに捨てられる。軽さは多くの場面で恩恵だ。だが軽い
意味には、立ち止まる契機がない。重さとは ―― ここで言う重さとは
質量ではなく、**そこで一度、手が止まること**だ。

人類は重要な意味に、ずっと**儀式**を与えてきた。署名する。読み
上げる。立会人を置く。儀式は意味に内容を足さない。意味を**重く
する**。手を止めさせ、見させ、引き受けさせる。重さとは、意味に
対する儀式性そのものである。

Sens の可視化を、私はこの系譜に置きたい。意味を見えるようにする
とは、意味に再び儀式性を ―― 立ち止まる契機を ―― 与えることだ。
レントゲンは骨を治さない。だが医師と患者を、像の前で一度、立ち
止まらせる。可視化の本質的な働きは、修正ではなく、**注視を強いる
こと**にある。

## 8. 事前の儀式と、事後の発見

ここに、意味との関わり方の二つの様式を立てる。**事前儀式**と
**事後発見**である。

事後発見は、いまの LLM 対話の既定様式だ。まず生成させ、出力を
読み、「ああ、こういう意味だったのか」と後から発見する。意味は
行為のあとに掘り出される ―― 考古学のように。便利だが、意味は
すでに起きてしまったあとにしか手に入らない。

事前儀式は逆を行く。行為の前に意味を据え、見て、引き受け、
そのうえで進む。意味は行為に**先立って**構成される。Sens が
意味記述レイヤーを LLM の**外側**かつ**手前**に置くのは、
この様式を可能にするためだ。生成された文字列から意味を逆算する
のではなく、意味を先に記述し、見て、Diff を取り、選んで注入し、
モデルを越えて持ち越す。意味を、事後の発掘物から、事前の対象へ
引き戻す。

これは速さの主張ではない。**順序**の主張である。意味が行為の前に
来るのか、後に来るのか。Sens は前に置く側に立つ。

## 9. 深さは、説明されない

最後に、このエッセイ自身に関わる原則を一つ。

深さは説明できない。説明された深さは、もう深さではなく、深さに
ついての要約だ。レントゲンは「ここに構造があります」と説明しない。
像を出す。見た者が、深さを**体感**する。説明と体感の差は小さく
ない ―― 説明は知識を足し、体感は立ち位置を変える。

Sens が機能の列挙で自らを語らないのは、レトリックの選択ではなく、
主張と一貫した態度だ。意味を扱うレイヤーは、意味を見せることで
証明されるべきで、意味について語ることで証明されるのではない。
だからこのエッセイの正しい読後感は「機能を理解した」ではなく、
**「これは別のカテゴリの話だった」と立ち位置が一段ずれること**で
ある。

レントゲン以前、人は骨折を**信じて**いた。レントゲン以後、人は
骨折を**見て**いる。同じ事実だ。変わったのは、人が立っている
場所のほうだ。Sens が問うているのは、機能ではなく、その場所で
ある。
