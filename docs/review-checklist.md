# Sens Papers — Review Checklist (ecri レビュー記録用)

> **用途**: Papers の成果物・更新を ecri が**逐次レビュー**し、リアクションを記録する。
> **更新**: Papers thread が新成果ごとに項目追加。ecri がチェックボックスで反応を記録。
> **最終更新**: 2026-06-13

## 使い方（凡例）

各項目に 3 つのリアクション・チェックボックス。該当する 1 つを `[x]` にし、必要なら「メモ:」に一言。

- **👍 承認** = このまま進めてよい
- **🔄 要修正** = 直してほしい（メモに要点）
- **❓ 質問/保留** = 確認したい・判断保留（メモに内容）

ファイル参照は repo ルート相対。`_` 始まりは gitignore 済ローカル WIP（push されない）。
