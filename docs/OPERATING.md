# Operating Notes (public)

This repo holds **public** Sens writing only.

Contributors MUST NOT include:
- Unreleased internal roadmap, timelines, or planning detail
- Any cost, budget, pricing, or economics figures
- Infrastructure internals, deployment topology, or operational tooling
- Hardware procurement decisions
- Unpublished experiment specifics or internal metrics
  - **Multimodal extension (2026-05-29, MM-P3 approved)**:
    - Image SAE training data IP (copyright / privacy / dataset
      provenance information)
    - Audio SAE training data IP (same; including speaker-identifiable
      material)
    - Shared embedding space internal research material (Z-approach
      research progress, unpublished implementation notes, internal
      benchmarks)
- Anything not already publicly stated by the project

Only intentionally-public material (conceptual / positioning /
philosophical narrative, publicly-cited literature) belongs here.

**The maintainer reviews every item before it leaves `drafts/` or is
published (site / GitHub / arXiv).** When in doubt, ask the maintainer.
