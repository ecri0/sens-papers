### 2.2 Papers の予備評価 (Q3-extended (b) B-2)

PRH 仮説と多モダリティ非依存 Feature 仮説は **概念的同型**:
- PRH = 異モデル間の表現収束 = 表現基底の差異を越えた意味の不変性
- 多モダリティ非依存 Feature = テキスト/音声/画像を越えた共通の意味特徴
- → 両者とも **「表現の差異 < 意味の不変性」** という同型の主張構造

2025 反例の条件分岐構造は、多モダリティ仮説に対しても「無条件には成立しない」
の相同予測を含意し得る (詳細: `sens-papers:handover/dispatch-to-dev-2026-05-29-q3-extended-detail.md` §3.2)。

## 3. Manifesto thread への 4 依頼事項

### 依頼 M-1: Constitution 条文整合性レビュー 🟡

- モダリティ表現の差異を越えた共通性を許容するか / 制約するか

  と「モダリティ非依存 Feature」の関係

→ 整合性に懸念がある場合、Constitution 改訂が必要か Manifesto thread から評価。

### 依頼 M-2: 哲学的背景 (Polanyi / Merleau-Ponty / Gärdenfors) の Manifesto レベル位置づけ 🟡

下記三哲学は **PRH と多モダリティ仮説の両者にまたがる**理論候補:

| 哲学 | PRH との関係 | 多モダリティとの関係 |
|---|---|---|
| Polanyi 暗黙知 | personal knowledge の収束基盤 | モダリティを越えた身体的知識統合 |
| Merleau-Ponty 身体現象学 | 知覚の身体的基盤 → 表現不変性の現象学的位置 | 身体的 multi-sensory perception |
| Gärdenfors Conceptual Spaces | モダリティを越えた幾何的意味空間 | 多モダリティ Feature の表現空間候補 |

Manifesto thread への質問:
- これらを Manifesto の **語彙的支柱** (write-and-hold 中の `_manifesto-core.md`) に
  取り込む余地があるか
- Constitution の哲学的基盤として明示するか / Papers technical 領域に留めるか
- 三者の優先度 (Papers としては Gärdenfors が数学的 framework として最有望と
  予備評価)

### 依頼 M-3: `_manifesto-core.md` (Papers write-and-hold) との関係整理 🟡

Papers `drafts/_manifesto-core.md` は **Manifesto の語彙的支柱**として
write-and-hold 中 (Papers Q2 で write-and-hold 期間判断 ecri 待ち)。

Manifesto thread と Papers `_manifesto-core.md` の **役割分担** を明確化したい:

| 案 | 内容 | Papers 予備見解 |
|---|---|---|
| 案 α | `_manifesto-core.md` を「公開向け語彙化」、Manifesto thread 本体を「非公開思想」と分離 | クリーン、Papers の役割と整合 |
| 案 β | `_manifesto-core.md` を Manifesto thread の **mirror**、Papers は推敲のみ | Manifesto から派生する形を明示、ただし Papers thread の独自起草余地は減る |
| 案 γ | Manifesto と Papers `_manifesto-core.md` は **異なる粒度**（Manifesto = フル思想、Papers = 公開可能語彙の予約簿）として併存 | 現状追認に近い、運用負荷は中 |

→ 案 γ を予備推奨するが、Manifesto thread の判断を仰ぐ。
